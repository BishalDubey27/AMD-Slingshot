# NutriSmart - Food & Health App

A smart solution that helps individuals make better food choices and build healthier eating habits by leveraging user behavior and contextual inputs.

## Features

### 🎯 Personalized Goal Setting
- Set health goals (weight loss, muscle gain, maintain health, increase energy)
- Define daily calorie targets
- Specify dietary preferences (vegetarian, vegan, low carb, high protein)

### 📊 Smart Meal Tracking
- Quick meal logging with natural language input
- Automatic nutrition estimation based on food descriptions
- Real-time calorie and protein tracking

### 💡 Intelligent Recommendations
- Context-aware suggestions based on eating patterns
- Alerts when approaching calorie limits
- Protein intake optimization tips
- Vegetable consumption reminders
- Goal-specific guidance

### 📈 Daily Insights
- Today's calorie consumption
- Remaining calorie budget
- Protein intake tracking
- Meal count monitoring

### 📝 Meal History
- View recent meals with timestamps
- Track nutritional information per meal
- Persistent data storage using localStorage

## How It Works

1. **Set Your Goals**: Define your health objectives and dietary preferences
2. **Log Meals**: Enter what you eat in natural language (e.g., "grilled chicken salad")
3. **Get Insights**: View real-time nutrition tracking and progress
4. **Receive Recommendations**: Get personalized suggestions based on your eating patterns
5. **Build Better Habits**: Use data-driven insights to make healthier choices

## Technology Stack

- Pure JavaScript (ES6+)
- HTML5 & CSS3
- LocalStorage for data persistence
- Responsive design

## Getting Started

Simply open `index.html` in your web browser. No installation or build process required!

## Key Intelligence Features

- **Pattern Recognition**: Analyzes meal descriptions to estimate nutrition
- **Behavioral Insights**: Tracks eating patterns throughout the day
- **Contextual Recommendations**: Provides timely suggestions based on current intake
- **Goal Alignment**: Ensures recommendations match user's health objectives
- **Adaptive Feedback**: Adjusts guidance based on accumulated data
