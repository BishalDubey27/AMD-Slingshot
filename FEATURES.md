# NutriSmart - Feature Overview

## Core Features ✨

### 1. Smart Goal Setting 🎯
- **Weight Loss**: Calorie deficit tracking with balanced nutrition
- **Muscle Gain**: High protein recommendations and surplus calories
- **Maintain Health**: Balanced macro tracking
- **Increase Energy**: Meal timing and nutrient optimization

**Customization:**
- Set daily calorie targets
- Choose dietary preferences (vegetarian, vegan, low carb, high protein)
- Personalized recommendations based on goals

### 2. Natural Language Meal Logging 📝
**How it works:**
- Type what you ate in plain language
- No searching through databases
- Instant nutrition estimation

**Examples:**
- "chicken biryani" → 600 cal, 25g protein
- "2 idlis with sambar" → 200 cal, 8g protein
- "paneer sandwich" → 400 cal, 28g protein
- "grilled chicken salad" → 250 cal, 15g protein

**Supported Foods:**
- Indian: dosa, idli, biryani, dal, roti, chapati, paratha, samosa, pakora, paneer, curd, rice
- Western: burger, pizza, pasta, salad, sandwich, eggs
- Common: fruits, smoothies, chicken, fish, yogurt

### 3. Real-Time Insights Dashboard 📊
**Displays:**
- Total calories consumed today
- Remaining calorie budget
- Protein intake (grams)
- Number of meals logged

**Updates:**
- Instant refresh after each meal
- Visual progress indicators
- Goal-aligned metrics

### 4. AI-Powered Recommendations 🤖

#### Time-Based Recommendations
- **Morning (before 10 AM)**: Breakfast reminders
- **Midday (10 AM - 2 PM)**: Lunch suggestions
- **Evening**: Dinner timing guidance
- **Late meals**: Meal spacing alerts

#### Calorie-Based Recommendations
- **Approaching limit (80%+)**: Lighter meal suggestions
- **Under target**: Encouragement to eat more
- **Over target**: Balanced approach guidance

#### Macro-Based Recommendations
- **Low protein (<50g)**: Protein-rich food suggestions
  - Vegetarian: paneer, dal, chickpeas, tofu
  - Non-vegetarian: chicken, fish, eggs, legumes
- **Low vegetables**: Veggie intake reminders
- **Macro imbalance**: Balanced meal suggestions

#### Behavioral Nudges
- **Hydration**: Water intake reminders
- **Meal timing**: Regular eating pattern suggestions
- **Food variety**: Diverse nutrient source recommendations
- **Portion control**: Mindful eating tips

#### Goal-Specific Guidance
- **Weight loss**: Calorie deficit maintenance, metabolism warnings
- **Muscle gain**: Protein optimization, calorie surplus
- **Maintenance**: Balanced nutrition, consistency
- **Energy**: Meal timing, complex carbs

### 5. Meal History Tracking 📅
- View last 10 meals
- Timestamp for each entry
- Calorie count per meal
- Chronological order (newest first)

### 6. Privacy-First Design 🔒
**Zero Data Collection:**
- No signup or registration
- No email, phone, or personal info
- No tracking pixels or analytics
- No cookies or fingerprinting

**Local Storage:**
- All data stays on your device
- Browser LocalStorage API
- No server communication
- User controls all data

**Transparency:**
- Open source code
- Verifiable privacy (check network tab)
- Clear data handling

### 7. Offline Capability 📴
- Works without internet after initial load
- All processing happens locally
- No API dependencies for core features
- Fast and responsive

### 8. Responsive Design 📱
- Works on desktop, tablet, mobile
- Adaptive layout
- Touch-friendly interface
- Consistent experience across devices
